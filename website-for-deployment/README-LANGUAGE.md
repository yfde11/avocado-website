# Avocado AI 網站語言切換功能說明

## 概述

Avocado AI 網站現在支援中英文雙語切換功能，用戶可以根據需要選擇繁體中文或英文來瀏覽網站內容。

## 功能特點

- 🌐 **雙語支援**：支援繁體中文 (zh-TW) 和英文 (en)
- 💾 **記憶功能**：使用 localStorage 記住用戶的語言偏好
- 🔄 **即時切換**：點擊按鈕即可即時切換語言
- 📱 **響應式設計**：適配各種設備尺寸
- 🎨 **美觀界面**：現代化的語言切換按鈕設計

## 使用方法

### 1. 語言切換按鈕

在每個頁面的導航欄右上角，您會看到一個語言切換按鈕：
- 顯示「中文」：當前為繁體中文
- 顯示「EN」：當前為英文

### 2. 切換語言

點擊語言切換按鈕即可在兩種語言之間切換：
- 從繁體中文切換到英文
- 從英文切換到繁體中文

### 3. 自動記憶

系統會自動記住您的語言選擇，下次訪問網站時會使用您上次選擇的語言。

## 技術實現

### 核心文件

- `language-switcher.js` - 語言切換器核心腳本
- 各頁面 HTML 文件中的 `data-translate` 屬性

### 翻譯系統

使用 `data-translate` 屬性標記需要翻譯的元素：

```html
<h1 data-translate="hero.title">標題</h1>
<p data-translate="hero.subtitle">副標題</p>
```

### 語言切換器類別

```javascript
class LanguageSwitcher {
    constructor() {
        this.currentLang = 'zh-TW';  // 預設語言
        this.translations = { ... };  // 翻譯字典
    }
    
    toggleLanguage() { ... }      // 切換語言
    translatePage() { ... }       // 翻譯頁面
    saveLanguage() { ... }        // 儲存語言偏好
}
```

## 添加新翻譯

### 1. 在 HTML 中添加翻譯標籤

```html
<!-- 原本的 HTML -->
<h1>新標題</h1>

<!-- 添加翻譯標籤後 -->
<h1 data-translate="new.title">新標題</h1>
```

### 2. 在語言切換器中添加翻譯

```javascript
// 在 translations 對象中添加
'zh-TW': {
    'new.title': '新標題'
},
'en': {
    'new.title': 'New Title'
}
```

## 測試語言切換功能

我們提供了兩個測試頁面來驗證語言切換功能：

### 1. 單頁面測試 (`test-language.html`)
- 測試單一頁面的翻譯功能

- 提供翻譯範例和測試說明

### 2. 全站測試 (`test-all-pages.html`)
- 測試所有頁面的語言切換功能
- 提供所有頁面的快速連結
- 驗證跨頁面語言狀態一致性

### 測試步驟
1. 打開任意測試頁面
2. 點擊導航欄右上角的語言切換按鈕
3. 觀察頁面內容是否正確切換語言
4. 檢查頁面標題是否更新
5. 切換到其他頁面，確認語言狀態保持一致
6. 重新整理頁面，確認語言偏好是否被記住

## 注意事項

1. **JavaScript 必需**：語言切換功能需要 JavaScript 支援
2. **本地儲存**：語言偏好會儲存在瀏覽器的 localStorage 中
3. **頁面重新載入**：語言切換會即時生效，無需重新載入頁面
4. **SEO 友好**：使用 `lang` 屬性標記頁面語言

## 故障排除

### 語言切換按鈕不顯示
- 檢查 `language-switcher.js` 是否正確載入
- 確認導航欄結構是否正確
- 如果按鈕仍然不顯示，頁面會自動重試創建按鈕
- 某些使用 React 的頁面可能需要額外的初始化時間

### 語言狀態不一致
- 所有頁面都會自動同步語言狀態
- 語言偏好會儲存在瀏覽器的 localStorage 中
- 重新整理頁面後語言狀態會保持一致

### 翻譯不生效
- 檢查 `data-translate` 屬性是否正確設置
- 確認翻譯鍵值是否在 `translations` 對象中定義

### 語言偏好不記憶
- 檢查瀏覽器是否支援 localStorage
- 確認瀏覽器設定是否允許本地儲存
