資安分析工程師

與我們一起打穿預設防線，讓資安防禦更強韌

【必要條件】

具備使用 Wireshark 或等效工具進行封包觀察與簡易分析的能力 / 曾使用 Windows Event Viewer 的使用與事件分析

【加分條件】

具備撰寫 Snort / Suriata 等 IDS 規則經驗 / 曾經操作過至少一種 SIEM 平台（如 ELK、Wazuh、Splunk…）

【工作內容】

監控、分析 Windows Event Viewer 日誌事件
透過 Wireshark 或其他工具觀察與分析封包流量，撰寫 snort 規則
使用 Wazuh 平台進行日誌分析與威脅偵測

【工控資安助理 / OT 安全實習生 / XDR 實習生】

強化下一代跨平台偵測與回應系統 ThreatCado XDR
推動智慧工廠打造 AI 驅動的防線

【工作內容】

測試與優化 ThreatCado XDR 系統模組（Log 解析、自動通報、Threat Graph）
撰寫針對 OT 場域的資安建議與 SOP
協助 SBIR、工業合作場域之專案進行
整合多來源情資（Wazuh、EDR）至分析平台

【AI Training 實習生 / 工程助理】

打造我們的 AI 資安助手 SenseL 的大腦與記憶體

【必要條件】

資訊工程、資訊管理或相關科系背景

具備前端與後端開發經驗，並熟悉以下任一後端框架：
- Flask
- FastAPI
- Node.js

熟悉以下工具與環境操作：
- Postman（API 測試）
- Docker（容器化部署）
- Linux 作業環境
- Git 版本控制工具
